1. Скопировать репозиторий на пк

перейти в папку в которую нужно скопировать
git clone ссылка на репозиторий

2.создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

создать новый репозиторий на гит
git add название файла (либо для всех новых файлов git add.)
git commit -m "Комментарий"
git push

3.получить на локальный компьютер изменения с github.com

git pull


Какую информацию нам дают команда :
git status - показывает состояние рабочего каталога
git log - показывает историю коммитов
git remote -v - показывает путь к репозиторию (доступные удалённые соединения)
